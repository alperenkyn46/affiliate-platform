# Affiliate Landing Page Platform

Modern, conversion odaklı affiliate reklam platformu.

## Teknoloji Stack

- **Frontend**: Next.js 14, TailwindCSS
- **Backend**: Node.js, Express.js
- **Database**: MySQL

## Kurulum

### Gereksinimler

- Node.js 18+
- MySQL 8+

### Adımlar

```bash
# Bağımlılıkları yükle
npm install

# Frontend geliştirme sunucusu
npm run dev:frontend

# Backend geliştirme sunucusu
npm run dev:backend

# Her ikisini birlikte çalıştır
npm run dev
```

## Klasör Yapısı

```
├── frontend/          # Next.js uygulaması
├── backend/           # Express.js API
├── database/          # SQL şemaları
└── README.md
```
